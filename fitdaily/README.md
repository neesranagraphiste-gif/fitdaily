# 💪 FitDaily — Sport & Nutrition

Application de sport quotidien avec suivi calorique. Fonctionne comme une app native sur mobile (PWA).

---

## 🚀 Déployer sur Vercel (10 minutes)

### Étape 1 — Créer un compte Vercel
Allez sur [vercel.com](https://vercel.com) → **Sign Up** → choisissez **Continue with GitHub** (créez un compte GitHub gratuit si vous n'en avez pas).

### Étape 2 — Importer le projet
1. Sur Vercel, cliquez **Add New → Project**
2. Cliquez **"Import Third-Party Git Repository"** ou utilisez le **drag & drop**
3. **Option la plus simple** : glissez-déposez le dossier `fitdaily` directement sur la page Vercel
4. Vercel détecte automatiquement Vite → cliquez **Deploy**

### Étape 3 — Récupérer votre lien
Après 1-2 minutes, Vercel vous donne un lien du type :
`https://fitdaily-xxxx.vercel.app`

---

## 📱 Installer sur votre téléphone (sans App Store !)

### Sur iPhone (Safari) :
1. Ouvrez le lien dans **Safari**
2. Appuyez sur le bouton **Partager** (carré avec flèche vers le haut)
3. Faites défiler → **"Sur l'écran d'accueil"**
4. Appuyez **Ajouter** → l'app apparaît comme une vraie app !

### Sur Android (Chrome) :
1. Ouvrez le lien dans **Chrome**
2. Appuyez sur les **3 points** en haut à droite
3. **"Ajouter à l'écran d'accueil"**
4. Appuyez **Ajouter** → icône sur votre écran !

---

## ✨ Fonctionnalités

- 🏋️ Programme 7 jours (haut du corps, bas, HIIT, core, full body, récup, repos)
- ⏱️ Timer de repos automatique entre les sets
- 🥗 Suivi calorique avec base de 20 aliments
- 📊 Calcul automatique BMR / TDEE / objectif perte de poids
- 📱 Fonctionne hors-ligne (PWA)
- 🔥 Compteur de streak
